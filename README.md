# 📊 Data Professional Survey Breakdown Dashboard

1. Overview

The **Data Professional Survey Breakdown Dashboard** is an interactive Power BI dashboard designed to analyze survey responses from data professionals across multiple countries. It provides insights into salary trends, job roles, programming language preferences, work-life balance, and career satisfaction to better understand the current data analytics landscape.

---

2. Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 **Power BI Desktop** – Dashboard development and data visualization.
- 📂 **Power Query** – Data cleaning, transformation, and preparation.
- 🧠 **DAX (Data Analysis Expressions)** – Calculated measures and KPIs.
- 📈 **Data Modeling** – Relationships between tables for accurate analysis.
- 📁 **Microsoft Excel** – Source dataset containing survey responses.

---

3. Data Source

**Source:** Microsoft Excel Survey Dataset

The dataset contains survey responses from **261 data professionals**, including information such as:

- Country
- Job Title
- Annual Salary
- Favorite Programming Language
- Age
- Work-Life Balance Rating
- Salary Satisfaction
- Difficulty in Breaking into Data

---

4. Features & Highlights

## Business Problem

Understanding salary trends, career satisfaction, and skill preferences in the data industry is essential for students, job seekers, recruiters, and organizations. Raw survey data is difficult to interpret without interactive visualizations.

## Goal of the Dashboard

The dashboard was created to:

- Analyze salary distribution across different data roles.
- Compare programming language preferences.
- Evaluate work-life balance and salary satisfaction.
- Understand the perceived difficulty of entering the data profession.
- Present survey insights through interactive visualizations.

---

5. Dashboard Walkthrough

### 📌 Key KPIs

- Total Survey Respondents
- Average Age of Participants

### 💰 Average Salary by Job Title

Compares the average salary across different data-related roles including Data Scientist, Data Engineer, Data Analyst, Data Architect, and Database Developer.

### 🌍 Country Distribution

Visualizes survey participation across different countries using a treemap.

### 💻 Favorite Programming Languages

Displays the programming languages most preferred by data professionals based on job roles.

### 📊 Difficulty to Break into Data

Illustrates respondents' opinions regarding how difficult it is to enter the data industry.

### 😊 Happiness Analysis

Gauge charts display:

- Work-Life Balance Rating
- Salary Satisfaction Rating

---

6. Key Insights

- Data Scientists reported the highest average salary among surveyed roles.
- Python is the most preferred programming language.
- The United States contributed the highest number of survey responses.
- Participants reported higher satisfaction with work-life balance than salary.
- Many respondents considered entering the data field to be moderately challenging.
